# Playwright Automation Practice And Javascript Program

This folder contains:
- **JavaScript practice programs** for QA Engineer role
- **Playwright with JavaScript practice programs** for test automation

## Overview
A comprehensive collection of JavaScript programs and Playwright automation scripts designed for QA engineers to practice and improve their skills in test automation and web testing.

## Contents
- JavaScript programs and utilities
- Playwright test automation examples
- Test configurations and setup files
- Data files for testing

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run tests: `npx playwright test`

## Technologies Used
- JavaScript (ES6+)
- Playwright
- Node.js
